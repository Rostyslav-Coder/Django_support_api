Код урока повторен и интегрирован в мой SUPPORT-API проект.
Интегрирован метод HTTP GET /api/pokemons/<str:name> -> Выбирает покемона из API
и сохраняет его в кеш моего SUPPORT-API.
Интегрирован метод HTTP GET /api/pokemons/ -> Возвращает всех покемонов,
доступных в переменной кеша.
Интегрирован метод HTTP DELETE /api/pokemons/<str:name> -> Удаляет покемона из кеша
моего SUPPORT-API, и возвращает удалённого покемона в качестве ответа на запрос.
В репазиторий добавлена коллекция Postman запросов, содержащих X-CSRF Token
для методов DELETE, и экспортированных в JSON формат.
На случай, если ключ X-CSRF Token изменится, в коллекцию добавлен пустой запрос,
возвращающий новый ключь X-CSRF Token в файлах COOKIE
